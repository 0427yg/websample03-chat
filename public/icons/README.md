# アイコン画像の配置

PWA として動作させるために、以下のサイズのアイコン画像を作成してこのフォルダに配置してください。

## 必要なアイコンサイズ

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## アイコンの作成方法

### オプション1: オンラインツールで作成（最も簡単）

1. [Canva](https://www.canva.com/) にアクセス
2. カスタムサイズで 512x512 の画像を作成
3. 💬 アイコンや「ChatConnect」のロゴをデザイン
4. PNG でダウンロード
5. [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator) にアップロード
6. 全サイズのアイコンを自動生成してダウンロード
7. このフォルダに配置

### オプション2: 簡易プレースホルダー（テスト用）

開発中は単色の PNG ファイルでも OK：

1. ペイントなどで 512x512 の画像を作成
2. 背景を #1e293b（ダークブルー）に設定
3. 中央に「💬」や「C」などを描画
4. 上記の各サイズにリサイズして保存

### オプション3: Figma / Photoshop でプロフェッショナルに作成

デザインツールで正方形アイコンを作成し、各サイズにエクスポート。

---

**注意:** 
- アイコンは正方形（1:1）で作成してください
- 透過PNG または白背景を推奨
- 最低限 192x192 と 512x512 があれば動作します
